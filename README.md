# Rock Paper Scissors

Small browser game built with HTML and JavaScript as part of The Odin Project. Play against the computer and track the score each round.

## Features

- Click-based game controls
- Random computer choice each round
- Score tracking
- Simple, clean UI

## How to Run

1. Open the project folder.
2. Launch `index.html` in your browser.

## How to Play

- Click Rock, Paper, or Scissors.
- The computer makes a random choice.
- The winner of the round is shown and the score updates.

## Tech Stack

- HTML
- JavaScript (vanilla)

## Project Structure

- `index.html`: Markup and game layout
- `logic.js`: Game logic and DOM updates
- `README.md`: Project overview

## Learning Goals

- Practice DOM manipulation
- Work with events and user input
- Use basic control flow and functions
- Keep UI in sync with game state

## Possible Improvements

- Add keyboard controls
- Add round history
